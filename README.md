# Jobpal

AI-powered job application assistant built for the Encode hackathon.

Keep one master profile, tailor your resume to any job in seconds, browse live job alerts, and track applications automatically from your inbox.

## Stack

Next.js 15 · Supabase · Azure AI Foundry · TypeScript

## Run locally

```bash
npm install
npm run dev
```

Copy `.env.example` to `.env.local` to connect real services. Everything works without any keys via built-in fallbacks.
